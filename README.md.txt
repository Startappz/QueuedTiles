Queued Tiles
=======
This project demonstrates how to add queued tiles to your windows 8 and windows phone project.

Queued Tiles
---
A queue of tiles that is changed automatically.

Run
---
```
TilesManager.ShowQueuedTiles();
```

Operating system requirements
---

- Windows 8.1
- Windows Phone 8.1
